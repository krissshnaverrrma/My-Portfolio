# 🧠 The Virtual AI Assistant Core (AI Logic)

The defining feature of this ecosystem is the native Virtual AI Assistant. Rather than forcing visitors to hunt through pages, I engineered a conversational interface powered by Google Gemini AI to act as my digital proxy.

**Context-Aware Prompt Engineering:** The LLM is injected with a comprehensive, hidden system prompt detailing my education, tech stack, project history, and professional philosophies. It acts as an interactive, real-time resume.

**State Management Architecture:** The UI features a dynamic status indicator reflecting the system's current operational mode:
**🟢 Online Mode:** Live Gemini API connection.

**🟡 Cache Mode:** Serving frequent queries from local memory.

**🟠 Database Mode:** Fetching archived responses.

**🔴 Offline Mode:** Graceful degradation when APIs are unreachable.

**Frictionless UX (Quick Actions):** Pre-configured query modules (e.g., "What is his tech stack?", "Show me his contact info") allow recruiters to extract specific data points instantly without typing.
